* eic.el
`eic.el` helps you capture English In Context.
`eic.el` was inspired by [[https://github.com/EvanMeek/shengci.el][shengci.el]] writen by EvanMeek.

* Description
`eic.el` captures words and its context for you with `eic/capture-word`.

You can set it up like this:

1. with leaf
#+begin_src elisp
  ;; eic
  (load "~/eic/eic.el")
  (leaf eic
    :bind
    ("C-c d c" . eic/capture-word))
#+end_src

2. or
#+begin_src elisp
  (load "~/eic/eic.el")
  (require 'eic)
  (global-set-key (kbd "C-c d c") 'eic/capture-word)
#+end_src
